# 🐾 SkinVet

Sistema web para gerenciamento de atendimentos em dermatologia veterinária.

O projeto foi desenvolvido com foco em aprendizado prático de desenvolvimento fullstack, mas também com aplicação real para uso em uma clínica veterinária.

---

## 🚧 Status do Projeto

⚠️ Em desenvolvimento contínuo

O sistema já possui funcionalidades principais implementadas e funcionando, porém está sendo constantemente atualizado com melhorias, novas features e ajustes.

---

## ✨ Funcionalidades

👤 Usuário:
- Cadastro e login
- Cadastro de pets
- Visualização dos pets cadastrados
- Agendamento de consultas
- Acompanhamento do histórico do pet

🐶 Pet:
- Cadastro com informações básicas (nome, espécie, raça, idade, peso)
- Upload de foto
- Histórico de acompanhamento dermatológico

📅 Agendamentos:
- Criação de agendamento
- Visualização de agendamentos

🩺 (Em desenvolvimento)
- Painel administrativo
- Gestão de atendimentos (aceitar/recusar)
- Relatórios (pendentes, confirmados, cancelados)
- Acompanhamento completo dos pets pela veterinária
